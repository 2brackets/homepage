2brackets.com homepage 

## Clone git repo

```bash
git clone https://github.com/2brackets/homepage.git
``` 

## Go to that folder

```bash
cd homepage 
``` 

## Install Dependencies

```bash
yarn install 
```

## Run on Local

```bash
yarn start
```

go to your browser and type http://localhost:3000
